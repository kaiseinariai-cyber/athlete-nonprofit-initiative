# Athlete Health Initiative Website

This is a simple static website that can be deployed directly on Vercel.

## Files
- `index.html` — main website page
- `styles.css` — styling
- `vercel.json` — optional Vercel config

## How to deploy on Vercel
1. Upload all files in this folder to your GitHub repository.
2. Go to Vercel.
3. Import the GitHub repository.
4. Vercel should detect it as a static site.
5. Click **Deploy**.

## Before publishing
Open `index.html` and replace:
- the placeholder email
- the placeholder Instagram handle
- the form links
